# 可一键还原的系统镜像备份指南

您的 Windows 10 系统缺少 `wbadmin`，且 DISM 无法在系统运行时直接备份 C 盘。以下为**推荐方案**和**备选方案**。

---

## 方案一：Macrium Reflect Free（推荐）

**免费、支持一键备份与还原，适合普通用户。**

### 1. 下载安装

- 官网：https://macrium.org/
- 选择 **Macrium Reflect Free** 下载
- 安装时选择免费版

### 2. 创建系统镜像

1. 打开 Macrium Reflect
2. 选择 **C 盘**（系统盘）
3. 点击 **Create an image of the partition(s) required to backup and restore Windows**
4. 目标位置选择：`F:\互娱电脑C盘备份\SystemImage\`
5. 点击 **Finish** → **OK** 开始备份

### 3. 创建救援启动盘（重要）

1. 在 Macrium Reflect 中点击 **Other Tasks** → **Create Rescue Media**
2. 选择 **Create Rescue Media**，按向导创建 **USB 启动盘**
3. 完成后，将 USB 插入电脑即可在无法进入系统时启动

### 4. 一键还原步骤

1. 插入存有系统镜像的 F 盘和救援 USB
2. 重启电脑，从 USB 启动（按 F12/F2/Del 选择启动项）
3. 进入 Macrium 救援环境后，选择 **Restore** → 选择镜像文件
4. 选择目标磁盘为 C 盘，按提示完成还原

## 方案二：Windows 自带工具（需从 WinPE 启动）

使用系统自带 DISM，但需先制作 **Windows PE 启动 U 盘**，再从 U 盘启动后备份。

### 1. 制作 WinPE 启动盘

1. 在另一台电脑或本机下载 **Windows ADK**：  
   https://learn.microsoft.com/zh-cn/windows-hardware/get-started/adk-install
2. 安装时勾选 **部署工具** 和 **Windows 预安装环境**
3. 以管理员身份打开 **部署和映像工具环境**
4. 执行：
   ```cmd
   copype amd64 C:\WinPE_amd64
   MakeWinPEMedia /UFD C:\WinPE_amd64 F:
   ```
   （将 F: 换成你的 U 盘盘符）

### 2. 从 WinPE 启动并备份

1. 用 WinPE U 盘启动电脑
2. 在命令提示符中确认 C 盘盘符（可能不是 C:）
3. 执行：
   ```cmd
   dism /Capture-Image /ImageFile:"X:\Backup.wim" /CaptureDir:C:\ /Name:"SystemBackup" /Compress:max /CheckIntegrity
   ```
   （将 X: 换成 F 盘或 U 盘盘符）

### 3. 还原步骤

1. 用 WinPE U 盘启动
2. 使用 `diskpart` 对目标磁盘分区
3. 执行：
   ```cmd
   dism /Apply-Image /ImageFile:"X:\Backup.wim" /Index:1 /ApplyDir:C:\
   bcdboot C:\Windows /s C:
   ```

## 重要提醒

1. **备份前**：关闭杀毒软件、尽量关闭其他程序，保证 F 盘空间充足（建议至少 150GB）
2. **备份后**：确认 `F:\互娱电脑C盘备份\` 下已有镜像文件
3. **重装前**：务必先完成系统镜像备份，再执行重装
4. **救援盘**：使用 Macrium 或傲梅时，一定要制作救援启动盘，否则系统崩溃时无法还原

## 备份进度计算（正确指标）

**应使用 `F:\互娱电脑C盘备份` 目录的实际大小** 作为进度指标，而非「F 盘空间减少」。

### 正确公式

```
进度 = min(100%, 备份目录实际大小 ÷ C盘已用空间 × 100%)
```

### 获取备份目录大小的命令

```powershell
# 方法：用 dir 统计（需约 2 分钟）
cmd /c "dir /s `"F:\互娱电脑C盘备份`" 2>nul"
# 输出末尾的「字节」数即为实际大小
```
